### 推荐配置 1：极简现代风（适合终端洁癖用户）
```bash
# Minimal Modern PS1 (white + blue + green)
PS1='\[\e[38;5;255m\]\u\[\e[38;5;33m\]@\[\e[38;5;46m\]\h\[\e[38;5;245m\]:\[\e[38;5;111m\]\w\[\e[38;5;255m\]$(__git_ps1 " (%s)")\[\e[38;5;255m\]\$\[\e[0m\] '
```
#### 效果说明：
- 用户名（白色）@ 主机名（蓝色）: 工作目录（深绿色）
- Git 分支（白色，带括号，仅在 Git 仓库显示）
- 权限符号：普通用户 `$`（白色）/root `#`（红色自动适配）
- 配色低饱和不刺眼，适合长时间使用
## 推荐配置 2：功能全面风（适合开发者 / 运维）
#### 效果说明：
- 左侧时间戳（青色，`HH:MM:SS` 格式，便于日志追溯）
- 用户名（亮绿色）@ 主机名（亮蓝色）: 工作目录（亮紫色）
- Git 分支（亮青色，带中括号）
- 权限符号：root 显示 `#`（亮红色），普通用户显示 `$`（亮红色）
- 所有文字加粗，层次感强，信息密度高
```bash
# Feature-Rich PS1 (with timestamp + Git status + permission)
PS1='\[\e[1;37m\][\[\e[0;36m\]\D{%H:%M:%S}\[\e[1;37m\]] \[\e[1;32m\]\u\[\e[1;33m\]@\[\e[1;34m\]\h\[\e[1;37m\]:\[\e[1;35m\]\w\[\e[1;36m\]$(__git_ps1 " [%s]")\[\e[1;31m\]$([[ $EUID -eq 0 ]] && echo "#" || echo "$")\[\e[0m\] '
```
## 推荐配置 3：复古终端风（适合喜欢怀旧风格的用户）
```bash
# Retro Terminal PS1 (amber + cyan + dark gray)
PS1='\[\e[38;5;214m\]\u\[\e[38;5;87m\]@\[\e[38;5;231m\]\h\[\e[38;5;156m\]:\[\e[38;5;241m\]\W\[\e[38;5;87m\]$(__git_ps1 " <%s>")\[\e[38;5;214m\]\$\[\e[0m\] '
```
#### 效果说明：
- 用户名（琥珀色）@ 主机名（白色）: 工作目录（短路径，浅橙色）
- Git 分支（青色，带尖括号）
- 权限符号（琥珀色）
- 配色模仿老式终端，低饱和度，复古感拉满
- 路径使用 `\W`（仅显示当前目录名），适合长路径场景
## 自定义修改指南：
- 改变颜色：修改 `\e[38;5;XXXm` 中的 `XXX`（使用 [ANSI 256 色码表](https://www.lihaoyi.com/post/BuildyourownCommandLinewithANSIescapecodes.html#256-colors)）
    - 示例：`38;5;255m` = 白色，`38;5;46m` = 绿色，`38;5;196m` = 红色
- 调整格式：
    - `\u` = 用户名，`\h` = 主机名，`\w` = 完整路径，`\W` = 简化路径
    - `\D{%H:%M:%S}` = 时间戳，可改为 `\D{%Y-%m-%d %H:%M}` 显示日期
- 移除 Git 分支：删除 `$(__git_ps1 " (%s)")` 部分
